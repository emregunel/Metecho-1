# Path

**Communicates the progress of a particular process.**

## Overview

The **Path** component visually communicates to a user the progress, determined
by its respective pull request on GitHub, made on a particular **Task** or
**Epic**.

For **Tasks**, the progress indicators are labeled `Planned`, `In progress`,
`Test`, and `Merged`.

The active status of a **Task** or **Epic** is indicated by the color blue.
Completed statuses feature a checkmark in place of the text label.

When an **Epic** or **Task** pull request is `Merged` on GitHub, the `Merge`
label turns green.
