# RemoveUser

**Confirms removal of a user from an Epic.**

## Overview

The **RemoveUser** component is an
[SLDS Modal](https://react.lightningdesignsystem.com/components/modals/) used to
confirm the removal of a user from an **Epic** when that user has a **Task**
assigned to them.

The modal displays a list containing the GitHub avatar and username for each
user selected for removal.

Pressing the `Confirm` button closes the modal and removes the user from the
**Epic**. Pressing `Cancel` closes the modal.
