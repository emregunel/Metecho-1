# ConfirmDelete

**Confirms before deleting or refreshing an Org that has unretrieved changes.**

## Overview

The **ConfirmDelete** component is an
[SLDS Modal](https://react.lightningdesignsystem.com/components/modals/) used to
confirm the deletion of a **Dev Org** that has unretrieved changes.

The modal is also triggered when attempting to refresh a **Scratch Org** that
has unretrieved changes.
